# 🎗️ Meme Kanseri Destek Mobil Uygulaması

Bu proje, meme kanseri tanısı almış hastaların tedavi süreçlerini daha kolay yönetebilmeleri, uzmanlara danışabilmeleri ve günlük belirtilerini takip edebilmeleri amacıyla geliştirilmiş bir Android mobil uygulamasıdır.

## 🚀 Projenin Amacı
Hastaların doğru bilgiye hızlı ulaşmasını sağlamak, tedavi sürecindeki (kemoterapi vb.) yan etkileri ve belirtileri kayıt altına alarak doktor-hasta iletişimini güçlendirmek.

## ✨ Temel Özellikler
* **Kullanıcı Kimlik Doğrulaması:** Firebase destekli güvenli Kayıt Ol / Giriş Yap sistemi.
* **Belirti Yönetimi & Takvimi:** Hastaların günlük belirtilerini (Ağrı, Bulantı, Enfeksiyon vb.) takvim üzerinden kayıt altına alması.
* **Uzmana Sor:** Doktorlara/Uzmanlara yazılı mesaj veya **kronometreli ses kaydı** ile soru iletebilme altyapısı.
* **Kan Tahlili Takibi:** Düzenli kan değerlerinin (WBC, Hemoglobin) takip edilebileceği modül.
* **Hasta Deneyimleri & Bilgilendirme:** Hastalık hakkında doğru bilgilerin ve diğer hastaların tecrübelerinin (video/metin) paylaşıldığı eğitim modülü.
* **Hızlı İletişim:** Tek tıkla telefon araması ve e-posta gönderme entegrasyonu.

## 🛠️ Kullanılan Teknolojiler
* **Programlama Dili:** Kotlin
* **Kullanıcı Arayüzü (UI):** XML (Android View System)
* **Veritabanı & Backend:** Firebase Authentication (Kimlik Doğrulama)
* **Geliştirme Ortamı:** Android Studio
* **Mimari:** Standart Android Activity Lifecycle, Intent tabanlı navigasyon.
